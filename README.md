# ToDoApp

簡単なToDoリストアプリを作成してみました　　

とりあえず基本的なリストアプリと盛れる機能を盛り込んでみました　　

## 機能　　
* ToDOの登録
* リストからタスクの削除
* アプリを開いた際にToDoリストの件数の通知
